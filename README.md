## How to run it local?
1. Clone the repository:

```
$ git clone hhttps://github.com/Hugh-Dev/mon-tera
```

2. Go to the cloned directory:
```
$ cd mon-tera
```

## How to run it?

3. run:
```
python setup.py install or pip install -r requirements.txt
```
